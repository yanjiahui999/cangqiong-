# DESIGN.md - 暗黑破坏神不朽·苍穹工会活动

## 气质与意象

地狱裂隙边缘的哥特式集会。暗红色光芒从大地裂缝中涌出，空气中弥漫着硫磺与灰烬，远处是燃烧的暗黑城堡剪影。苍穹工会的战旗在烈焰风暴中猎猎作响，金色符文在黑暗中脉动发光。

## 视觉策略

- 背景：深黑至暗红渐变，叠加烟雾纹理和微弱的火焰粒子效果
- 图形语言：哥特式边框装饰、符文纹路、五芒星元素
- 卡片/区块：半透明暗色玻璃质感，边缘带金色发光线条
- 分隔线：使用符文装饰的金色细线

## 配色方案

| 用途 | 色值 | 意象来源 |
|------|------|----------|
| 深渊黑（背景主色） | `#0a0a0f` | 地狱深处的无尽黑暗 |
| 血焰红（主强调色） | `#8b1a1a` → `#c0392b` | 燃烧的地狱之火 |
| 暗金色（标题/装饰） | `#c9a84c` → `#f4d03f` | 远古符文的金属光泽 |
| 幽紫色（辅助色） | `#4a1942` → `#6c3483` | 暗影魔法的余韵 |
| 灰烬白（正文） | `#d4cfc4` | 燃烧后的灰烬余温 |
| 硫磺黄（高亮） | `#e67e22` | 裂隙中溢出的硫磺之光 |

## 字体排版

- 标题字体：使用 Google Fonts 中的 `Cinzel`（拉丁）+ `Noto Serif SC`（中文），哥特式衬线体
- 正文字体：`Noto Sans SC`，确保可读性
- 标题策略：大号、字间距加宽、配合金色渐变和文字阴影
- 正文策略：灰烬白色，行高 1.8，保证暗底阅读舒适度

## 动效与交互

- Hero 区域：背景缓慢视差滚动 + 火焰粒子浮动动画
- 符文装饰：微弱的呼吸式发光脉冲（opacity 0.3→0.8 循环）
- 卡片悬停：边框金光增强 + 轻微上浮（translateY -4px）
- 滚动进入：各区块 fade-in + slide-up，交错延迟
- 按钮：悬停时金色光晕扩散，点击时有火焰闪烁效果

## 设计禁忌

- 不要使用明亮、饱和度过高的色彩（如亮蓝、亮绿）
- 不要使用圆角过大的卡片（保持哥特式锐利感，圆角不超过 4px）
- 不要使用卡通/可爱风格的图标
- 不要使用纯白色背景或区块
- 不要使用现代扁平化的设计风格
- 避免过多动画导致页面卡顿，保持克制
